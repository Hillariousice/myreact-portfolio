# 🌐 My Portfolio

My personal developer portfolio showcasing projects, skills, and experience — built with Next.js and Tailwind CSS.

🔗 **Live Site:** [myreact-portfolio.vercel.app](https://myreact-portfolio.vercel.app)

## Overview

This portfolio serves as a central hub showcasing my work as a React Native & Frontend Engineer, including project case studies, skills, and ways to get in touch.

## Features

- 🎨 Clean, responsive design built with Tailwind CSS
- ⚡ Server-side rendering via Next.js for fast load times
- 📁 Project showcase with live demo links
- 📬 Contact section for direct outreach

## Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Language:** JavaScript

## Getting Started

```bash
git clone https://github.com/Hillariousice/myreact-portfolio.git
cd myreact-portfolio
npm install
npm run dev
```

Open `http://localhost:3000` to view it locally.
